# HomePick - AI 기반 부동산 시장 분석을 통한 맞춤형 매물 추천 플랫폼 (Frontend)

## 대모 페이지
https://fsclass-n.github.io/homepick/

## 프로젝트 개요
AI 기반 부동산 시장 분석을 통한 맞춤형 매물 추천 플랫폼은 사용자의 선호도와 부동산 시장 데이터를 분석하여 최적의 매물을 추천하는 웹 서비스입니다. 본 프로젝트는 프론트엔드 개발에 집중하여 사용자 경험을 최적화하고 직관적인 UI/UX를 제공하는 것을 목표로 합니다.

## 프로젝트 목표
- 사용자 맞춤형 매물 추천 시스템 구현
- 직관적이고 사용하기 쉬운 인터페이스 제공
- 반응형 웹 디자인을 통한 다양한 디바이스 지원
- 빠르고 효율적인 페이지 로딩 및 사용자 경험 개선

## 기획 의도
현대인들의 복잡한 생활 패턴과 다양한 주거 요구를 충족시키기 위해, AI 기술을 활용한 개인화된 부동산 매물 추천 서비스를 기획하였습니다. 기존 부동산 포털의 정보 과잉 문제를 해결하고, 사용자에게 정말 필요한 매물만을 선별하여 제공함으로써 의사결정 시간을 단축하고 만족도를 높이는 것을 의도합니다.

| 김서연 (관리자 페이지 / admin.html) | - 매물 등록 및 수정/삭제<br>- 사용자 관리 (회원 목록, 권한 설정)<br>- 통계 대시보드 (방문자 수, 매물 조회수)<br>- 공지사항 관리 | - 관리자 대시보드 UI 설계<br>- 데이터 테이블 및 페이징 기능 구현<br>- 차트 라이브러리를 활용한 통계 시각화<br>- Figma를 활용한 관리자 페이지 프로토타입 설계 |

## 기술 스택
- **HTML5**: 시맨틱 마크업 및 웹 표준 준수
- **CSS3**: Flexbox, Grid 레이아웃, 애니메이션, 반응형 디자인
- **JavaScript (ES6)**: 모듈화, 화살표 함수, 비동기 처리, DOM 조작
- **Bootstrap 5**: Flex Box 기반 반응형 프레임워크, 컴포넌트 라이브러리
- **Git/GitHub**: 버전 관리, 협업, 배포
- **Figma**: UI/UX 디자인, 프로토타이핑, 디자인 시스템

## UI 핵심 기술
- **반응형 디자인**: Bootstrap 5의 Grid 시스템을 활용한 모바일/태블릿/데스크톱 대응
- **메가 메뉴**: 헤더 호버 시 전체 서브 메뉴 표시 (CSS transition으로 부드러운 애니메이션)
- **카드 UI**: 매물 정보를 카드 형태로 직관적으로 표시
- **모달 팝업**: 페이지 이동 없이 매물 상세 정보 확인
- **탭 메뉴**: 마이페이지에서 카테고리별 콘텐츠 구분
- **로딩 애니메이션**: 데이터 로딩 시 사용자 피드백 제공
- **호버 효과**: 버튼, 카드 등 인터랙티브 요소에 시각적 피드백
